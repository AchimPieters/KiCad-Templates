Raspberry Pi® Zero W - Hat Template

This project template is the basis of an expansion board for the Raspberry Pi® Zero W

This base project includes a PCB edge defined as the same size as the Raspberry Pi® Zero W PCB with the connectors placed correctly to align the two boards. All IO present on the Raspberry-Pi board is connected to the project through the 0.1" expansion headers.
